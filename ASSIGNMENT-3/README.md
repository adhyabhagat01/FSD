
👗 Fashion Store Web Application
Assignment 3 – Full Stack Development
This project is a simple Fashion Store web application developed using HTML, CSS, JavaScript, and Bootstrap.
The application simulates an online fashion shopping interface where users can browse products, add them to the cart, and view the cart summary.
The main goal of this assignment is to understand front-end development concepts, responsive design using Bootstrap, DOM manipulation using JavaScript, and basic UI interaction.

Project Objectives
The objectives of this assignment are:
•	To design a responsive e-commerce interface
•	To implement navigation between different sections
•	To perform DOM manipulation using JavaScript
•	To implement a basic cart system
•	To practice UI styling using CSS and Bootstrap
•	To organize a web project with separate HTML, CSS, and JavaScript files

Technologies Used
The following technologies were used in this project:
Technology	Purpose
HTML5	Structure of the web page
CSS3	Styling and layout
Bootstrap 5	Responsive design and UI components
JavaScript (Vanilla JS)	Interactivity and cart functionality
