HTML Text Formatting Tags – Assignment 1

Objective
The objective of this assignment is to understand and implement HTML text formatting tags and display them in a structured format using an HTML table.

Project Overview
This project is a simple static web page created using HTML and CSS that demonstrates different HTML text formatting tags. The page displays these tags in a tabular format with their descriptions and examples, making it easier to understand how each tag affects text appearance in a webpage.
The webpage includes commonly used HTML formatting tags such as bold, italic, emphasized, highlighted, subscript, and superscript text.

What the Project Does
The project creates a structured webpage that:
•	Displays various HTML text formatting tags
•	Shows the description of each tag
•	Demonstrates how the tag appears in the browser using examples
•	Organizes all the information in a clean and readable table format

Why the Project is Useful
This project is useful for beginners learning HTML fundamentals because it:
•	Helps understand how text formatting works in HTML
•	Provides a visual demonstration of formatting tags
•	Shows how to use HTML tables to organize content
•	Demonstrates basic CSS styling for webpage design
It can also serve as a quick reference for commonly used HTML formatting tags.
