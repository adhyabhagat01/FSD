
Personal Portfolio Website
Assignment 2 – Full Stack Development

This project is a Personal Portfolio Web Application developed using HTML, CSS, JavaScript, and Bootstrap. The portfolio website showcases personal information such as projects, skills, achievements, certifications, and contact details in a professional and responsive layout.
The main goal of this assignment is to understand front-end development concepts, responsive design using Bootstrap, layout structuring using HTML and CSS, and basic JavaScript integration.

Project Objectives
The objectives of this assignment are:
• To design a personal portfolio website
• To implement responsive layout using Bootstrap
• To organize website content into different sections
• To practice HTML structure and CSS styling
• To integrate JavaScript with a web page
• To create a professional digital portfolio

Technologies Used
The following technologies were used in this project:
Technology	Purpose
HTML5	Structure of the web page
CSS3	Styling and layout design
Bootstrap 5	Responsive UI components
JavaScript	Basic functionality and script execution
